# Figures

Place article and README screenshots in this folder. Recommended filenames:

- `museum-overview.png`: Wide view of the 3D gallery.
- `welcome-modal.png`: Startup mode chooser.
- `artwork-modal.png`: Artwork detail modal with image, audio, or video.
- `credits-modal.png`: Credits overlay.
- `guided-tour.png`: Guided tour HUD or stop view.
- `performance-results.png`: FPS or profiling screenshot after measurement.

Keep screenshots current with the implemented UI and avoid including private browser or operating-system information.
